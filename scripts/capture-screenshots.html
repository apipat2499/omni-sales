üõ†Ô∏è Method 1: Browser Developer Tools

Using Chrome DevTools

  1. Open your app in Chrome: npm run dev then visit http://localhost:3000
  2. Press F12 to open Developer Tools
  3. Click the device toolbar icon (or press Ctrl+Shift+M)
  4. Select device dimensions:
    • For mobile: 540 x 960
    • For desktop: 1280 x 800
  5. Navigate to each page
  6. Press Ctrl+Shift+P and search for "Capture screenshot"
  7. Select "Capture full size screenshot"
  8. Save to public/screenshots/ folder

üõ†Ô∏è Method 2: Browser Extensions

Using Awesome Screenshot (Recommended)

  1. Install "Awesome Screenshot" extension for Chrome/Firefox
  2. Navigate to the page you want to capture
  3. Click the extension icon
  4. Select "Capture entire page"
  5. Use the built-in editor to crop to exact dimensions
  6. Save as PNG to public/screenshots/

Other Good Options

  • Fireshot: Full page screenshots with annotation
  • Nimbus: Video + screenshot capture
  • GoFullPage: One-click full page capture

üõ†Ô∏è Method 3: Playwright (Automated)

Install and Run Script

Create scripts/capture-screenshots.js:

const { chromium } = require('playwright');

async function captureScreenshots() {
  const browser = await chromium.launch();
  const context = await browser.newContext({
    viewport: { width: 1280, height: 800 }
  });
  const page = await context.newPage();

  const pages = [
    { url: 'http://localhost:3000/dashboard', name: 'dashboard' },
    { url: 'http://localhost:3000/products', name: 'products' },
    { url: 'http://localhost:3000/orders', name: 'orders' },
    { url: 'http://localhost:3000/customers', name: 'customers' },
    { url: 'http://localhost:3000/reports', name: 'reports' },
  ];

  for (const { url, name } of pages) {
    await page.goto(url);
    await page.waitForTimeout(2000); // Wait for charts to render
    await page.screenshot({
      path: `public/screenshots/${name}-desktop.png`,
      fullPage: false
    });
    console.log(`‚úÖ Captured ${name}`);
  }

  // Mobile screenshots
  await context.setViewportSize({ width: 540, height: 960 });
  await page.goto('http://localhost:3000/dashboard');
  await page.waitForTimeout(2000);
  await page.screenshot({
    path: 'public/screenshots/dashboard-mobile.png',
    fullPage: false
  });

  await browser.close();
  console.log('üéâ All screenshots captured!');
}

captureScreenshots();

Install and run:

npm install -D playwright
node scripts/capture-screenshots.js

‚úÖ Post-Capture Checklist

  1. Verify all screenshots are in public/screenshots/
  2. Check file sizes (should be < 500KB each)
  3. Ensure proper dimensions
  4. Update manifest.json with screenshot paths
  5. Test PWA install prompt shows screenshots
  6. Optimize images if needed (use TinyPNG or similar)
