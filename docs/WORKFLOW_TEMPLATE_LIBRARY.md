**Difficulty:** Beginner

### Email Sequence

**Email 1 - Welcome (Immediate)**

- CTA: Learn More

## Customization Tips

1. **Personalization**
   - Use customer name
   - Reference past purchases
   - Show browsing history
   - Include location data

2. **Timing**
   - Test different send times
   - Consider time zones
   - Avoid weekends for B2B
   - Optimize by segment

3. **Subject Lines**
   - A/B test variants
   - Use emojis sparingly
   - Create urgency
   - Ask questions

4. **Content**
   - Keep it scannable
   - Use clear CTAs
   - Include social proof
   - Add product images

5. **Mobile Optimization**
   - Responsive design
   - Large tap targets
   - Short subject lines
   - Concise content
